class storage.TextObjectStorage(txtobjtypes, primaryfile, files=[])¶

Bases: collections.abc.MutableSequence, watchdog.events.FileSystemEventHandler

Persistant storage of textobjects.TextObject subclasses abstracted as a mutable sequence

txtobjtypes¶

The TextObject subclasses which will be stored

Type

List[textobjects.TextObject]

primaryfile¶

the path to the primary storage file. Entries will be added to this file when a new item is added.

Type

str

files¶

the paths to any storage files. All occurances of the txtobjtypes in these files will show up in the sequence. When a textobject is updated the occurance of it in it’s respective file will be replaced.

Type
